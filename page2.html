Job Description:


We are looking for a skilled and passionate Front-End Developer to join our dynamic team! If you have a knack for creating visually appealing, responsive, and user-friendly web interfaces, this opportunity is for you.


Requirements to Apply:


  • A minimum CGPA of 7 or equivalent in your academic record.
  • Proficiency in HTML and CSS for creating structured and styled web pages.
  • Experience with JavaScript for adding interactivity to websites.
  • Familiarity with React for building modern web applications.
  • Understanding of Tailwind CSS for efficient styling and responsive design.

Desired Skills:


  • Strong problem-solving skills and an eye for detail.
  • Ability to work with RESTful APIs and integrate backend services.
  • Knowledge of version control systems like Git.
  • A passion for staying updated with the latest trends in front-end development.
